    Projet réalisé pendant le stage :

    Création d'un site de réservation d'appartement en ligne pour les salariés de l'entreprise qui se déplacent sur Paris.

    Objectif :
    L'objectif de ce projet est de mettre en place une application web de réservation d'appartements pour les collaborateurs de Prodware. Cette application doit permettre aux collaborateurs de réserver les deux appartements disponibles à Paris de manière autonome, tout en offrant aux administrateurs/gestionnaires la possibilité d'annuler les réservations si nécessaire.

    Exigences techniques :
    L'application a tout d'abord était crée via les outils PowerPlateform notamment PowerPages. Puis nous avons développé le site via Php et en utilisant une base de données MySQL

    Mise en place :
    Tout d’abord j'ai du faire valider le cahier des charges puis j’ai créé une première maquette visuel via l'outil figma. Enfin j’ai réalisé le projet a l’aide de PhpStorm et WampServer pour héberger la base de donnée.
